Golden Kubestronaut Golden Kubestronaut is a title given to active holders of all CNCF certifications, plus the Linux Foundations Certified System Administrator. At the time of writing this is a total of 14 certifications, with another one joining the ranks mid-October. This is a level up from the Kubestronaut title, which is just the Kubernetes focused certifications, the goal of the level up is to show knowledge across the larger Cloud Native Landscape. This post is not meant to be a sales-pitch for the programme, CNCF’s own page does that quite well, but I do get a backpack for obtaining this title. ...
